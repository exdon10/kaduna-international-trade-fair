## Style Guidelines:

- Primary color: Deep royal blue (#1E3A8A), to convey a sense of trustworthiness and stability.
- Background color: Very light blue (#F0F9FF), for a clean, professional feel.
- Accent color: Light cyan (#7DD3FC), used for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font providing a clean, modern, objective feel for readability and a professional look.
- Use simple, professional icons to represent sponsorship benefits and key information. 
- A clean, well-organized layout with clear sections for each sponsorship tier and benefits, emphasizing ease of navigation and a professional presentation.
- Subtle animations to highlight key information and calls to action, ensuring a dynamic yet professional user experience.
